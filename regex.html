Character classes
. any character except newline
\w \d \s word [a-zA-Z0-9_], digit, whitespace [\f\t\n\r ]
\W \D \S not word, digit, whitespace
\0 null character
[abc] any of a, b, or c
[^abc] not a, b, or c
[- \^] any of -, space, \ or ^
[a-g] character between a & g
[i-kxy-] any of i, j, k, x, y or -
[:alpha:] [:alnum:] [:digit:] posix classes
[:upper:] [:lower:] [:punct:] posix classes
\c \x \O control character, hex, octal digit
\p{Arabic} \P{Han} any Arabic, non CJK (ChineseJK) character
Quantifiers & Alternation
a* a+ a? 0 or more, 1 or more, 0 or 1
a{5} a{2,} exactly five, two or more
(abc)+ (abc){1,} abc one or more times
a{1,3} between one & three
a+? a{2,}? <div>.*?</div> match as few as possible ie nongreedy/lazy
ab|cd match ab or cd
Anchors
^abc$ start / end of the string
\Aabc\Z abc\z start / end / very end of string
\b \B word, not-word boundary
\<cat\> word in word boundary
Escaped characters .*+? ^$ \| ()[{
\. \* \\ escaped special characters
\t \n \r tab, linefeed, carriage return
\N \R not linefeed (P), one linebreak+\v
\v \h vertical, horiz whitespace
\V \H not vertical, horiz whitespace
\cC \x41 \077 \u00A9 control-c, hex, octal, unicode escaped Â©
\QC++\E treat these as literals
Groups & Lookaround
(abc) capture group
\1 backreference to group #1
\b(\w+) \1\b two identical words separated by space
(?:abc) non-capturing group
(?=abc) positive lookahead
(?!abc) negative lookahead
(?<=abc) positive lookbehind
(?<!abc) negative lookbehind
Replacement
$1 first capture group
$` $' before, after matched string
$+ last matched string
$& entire matched string
$$ $
Modifiers
(?s) (?m) single line, multiline mode
(?i) case insensitive
g x e U global, allow comments, evaluate, ungreedy
(?n) (?d) named capture(.NET), linebreaks only(Java)
\K keep out: drop everything matched
